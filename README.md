# Flask-Blog
A simple blog written with Flask framework.

## Installation
 
    $ git clone https://github.com/djeada/Minimal-Flask-App.git
    $ cd Minimal-Flask-App
    $ python3 -m venv myenv
    $ source myenv/bin/activate
    $ python3 src/app.py
